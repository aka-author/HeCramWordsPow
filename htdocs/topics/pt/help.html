Selecionando palavras

As seguintes opções estão disponíveis para selecionar palavras:

  • Uma parte do discurso
  • Domínios de assunto

Uma parte do discurso e um domínio de assunto fornecem uma interseção. Para exemplo, se Verbos e Culinária são selecionados ao mesmo tempo, então você obtém uma lista de verbos que se relacionam para cozinhar.

Alguns domínios de assunto fornecem uma união. Diga se Culinária e Férias são selecionados ao mesmo tempo, então você obtém uma lista de palavras relacionadas à culinária e também às palavras relacionadas aos feriados. Use este vocabulário para discutir uma próxima festa com seus amigos.

Para selecionar uma parte do discurso:

  • Selecione uma parte do discurso na Parte do discurso lista suspensa

Para desmarcar uma parte do discurso:

  • Selecione o item Todos em a lista suspensa Parte do discurso

Para selecionar um domínio de assunto:

  • Clique em uma tag de domínio de assunto desmarcada

Para desmarcar um domínio de assunto:

  • Clique em uma tag de domínio de assunto selecionado

Impressão de cartões

Para formatar palavras para imprimi-las em cartões:

  1. Selecione as palavras que você precisa para imprimir
  2. Torne a lista de palavras selecionadas visível, se estiver oculta
  3. Pressione o o botão
  4. Imprimir páginas com as palavras em um idioma base
  5. Vire as folhas de papel e coloque-as de volta na impressora
  6. Imprima páginas com as palavras em um idioma de destino
