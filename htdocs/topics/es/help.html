Seleccionar palabras

Las siguientes opciones están disponibles para seleccionar palabras:

  • Una parte del discurso
  • Dominios de materias

Una parte del discurso y un dominio temático dan una intersección. Para ejemplo, si Verbos y Cocina se seleccionan al mismo tiempo, entonces obtienes una lista de verbos que se relacionan a cocinar.

Algunos dominios de materias dan una unión. Di, si Cocina y Vacaciones se seleccionan al mismo tiempo, luego obtiene una lista de palabras relacionadas a la cocina y palabras relacionadas con las vacaciones. Usa este vocabulario para discutir una próxima fiesta con tus amigos.

Para seleccionar una parte del discurso:

  • Seleccione una parte del discurso en la Parte del discurso lista desplegable

Para anular la selección de una parte del discurso:

  • Seleccione el elemento Todo en la Parte del discurso lista desplegable

Para seleccionar un dominio temático:

  • Haga clic en una etiqueta de dominio de asunto no seleccionada.

Para anular la selección de un dominio de materia:

  • Haga clic en una etiqueta de dominio de materia seleccionada.

Impresión de tarjetas

Para formatear palabras para imprimirlas en tarjetas:

  1. Seleccione las palabras que necesita imprimir
  2. Hacer visible la lista de palabras seleccionadas si está oculta
  3. Pulsa el el botón
  4. Imprime páginas con las palabras en un idioma base
  5. Da la vuelta a las hojas de papel y devuélvelas a la impresora
  6. Imprima páginas con las palabras en un idioma de destino
